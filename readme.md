# Bomb Manual Assistant

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个专为合作解谜游戏《Keep Talking and Nobody Explodes》设计的智能辅助工具。本程序旨在帮助玩家（尤其是“专家”角色）更快速、精准地查询官方《Bomb Defusal Manual》，从而优化团队沟通，提升拆弹效率。

## ✨ 功能特性

- **无脑拆弹**：所有的支持模块只需要无脑输入相关参数，就会自动提示相应操作。
- **自动置顶**：程序会自动进行置顶，保证不会被其他窗口覆盖。提升使用效率。
- **界面清晰**：简洁直观的界面设计，确保“专家”在高压下也能快速获取信息，避免信息过载。
- **离线使用**：所有数据本地化，无需网络连接即可运行，确保游戏过程零延迟。

## 🚀 开始使用

### 前提条件
- 你需要一台运行 Windows 10 或更高版本的电脑。
- 你需要拥有游戏《Keep Talking and Nobody Explodes》。
- 一份官方的《Bomb Defusal Manual》（通常为PDF格式）。本项目不完整包含手册内容。
- 请注意，本程序需要 **管理员权限** 以确保置顶功能正常工作。

### 安装与运行
- 请直接在release页面下载最新版本的可执行文件：[Releases](https://github.com/123456789-code/Bomb-Manual-Assistant/releases)

## 📖 使用指南

1.  启动 `Bomb Manual Assistant`。
2.  **“拆弹员”** 在游戏中观察炸弹，并向 **“专家”** 描述模块特征。
3.  **“专家”** 在程序的中选择对应模块并输入参数，程序会立即做出判断。
4.  **“专家”** 根据程序提示的规则，指导拆弹员完成操作。
